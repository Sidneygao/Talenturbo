### 4. ✅ 两种实现方式

#### 方式1：CSS类（推荐）
```html
<span class="trademark">Talenturbo</span>
<!-- 显示为：Talenturbo® -->
```

优点：
- 统一管理
- 易于维护
- 可以全局调整样式

#### 方式2：HTML标签
```html
HR CORE<sup>®</sup>
<!-- 显示为：HR CORE® -->
```

优点：
- 直观明确
- 不依赖CSS
