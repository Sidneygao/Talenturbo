# 页面背景图片配置

## 背景方案

### 默认背景（CSS纹理图案）
- **类型**：纯CSS几何纹理
- **颜色**：浅灰色系 (#f8f9fa, #f0f2f5, #e8eaed)
- **图案**：六边形蜂窝纹理
- **特点**：
  - 无需加载图片，性能最优
  - 低调、现代、不抢眼
  - 完全响应式
  - 无版权问题
- **适用**：所有未指定特殊背景的页面

### 产品页面特殊处理
- **背景**：纯色 (#f8f9fa)，无纹理
- **原因**：每个产品区块都有自己的配图
- **效果**：让产品图片更突出

## 技术实现

### CSS纹理背景
```css
body.page {
  background: #f8f9fa;
  background-image: 
    linear-gradient(30deg, #f0f2f5 12%, transparent 12.5%, transparent 87%, #f0f2f5 87.5%, #f0f2f5),
    linear-gradient(150deg, #f0f2f5 12%, transparent 12.5%, transparent 87%, #f0f2f5 87.5%, #f0f2f5),
    /* ... 更多渐变层 */;
  background-size: 80px 140px;
  background-position: 0 0, 0 0, 40px 70px, 40px 70px, 0 0, 40px 70px;
}
```

### 产品页面背景
```css
body.page.products-page {
  background: #f8f9fa;
  background-image: none; /* 移除纹理 */
}
```

### 其他页面背景
```css
body.page.services-page {
  background-image: url('图片URL');
  background-size: cover;
  background-position: center;
  background-attachment: fixed;
}
```

## 背景遮罩层

### 默认遮罩（纹理背景）
```css
body.page::before {
  background: linear-gradient(180deg, 
    rgba(255,255,255,.85) 0%, 
    rgba(250,251,252,.90) 100%);
}
```
- **透明度**：85%-90%（更高，因为纹理已经很淡）

### 产品页面遮罩
```css
body.page.products-page::before {
  background: linear-gradient(180deg, 
    rgba(255,255,255,.75) 0%, 
    rgba(250,251,252,.82) 100%);
}
```
- **透明度**：75%-82%（标准）

## 自定义纹理图案

### 修改颜色
```css
/* 将 #f0f2f5 和 #e8eaed 替换为你想要的颜色 */
background-image: 
  linear-gradient(30deg, #你的颜色 12%, transparent 12.5%, ...);
```

### 修改图案大小
```css
/* 调整 background-size 的值 */
background-size: 80px 140px; /* 改为 100px 175px 会更大 */
```

### 其他纹理图案

#### 点阵图案
```css
background-image: radial-gradient(circle, #e8eaed 1px, transparent 1px);
background-size: 20px 20px;
```

#### 条纹图案
```css
background-image: repeating-linear-gradient(
  45deg,
  #f0f2f5,
  #f0f2f5 10px,
  #e8eaed 10px,
  #e8eaed 20px
);
```

#### 网格图案
```css
background-image: 
  linear-gradient(#e8eaed 1px, transparent 1px),
  linear-gradient(90deg, #e8eaed 1px, transparent 1px);
background-size: 20px 20px;
```

## 响应式处理

### 移动端优化
```css
@media (max-width: 768px) {
  body.page.services-page,
  body.page.cases-page,
  body.page.partners-page,
  body.page.about-page {
    background-attachment: scroll;
  }
}
```

- **纹理背景**：无需特殊处理，自动响应
- **图片背景**：移动端改为scroll，提升性能
