# 问题排查指南

## 问题 1：LOGO 淡蓝色背景仍然存在

### 原因分析
LOGO.jpg 图片文件本身可能就包含淡蓝色背景。CSS 滤镜只能部分减轻，无法完全去除。

### 解决方案（按优先级）

#### 方案 A：替换为透明背景的 PNG 图片（推荐）
1. 使用图片编辑软件打开 LOGO.jpg
2. 去除背景，保存为 PNG 格式（支持透明背景）
3. 替换文件或重命名为 LOGO.png
4. 更新所有 HTML 文件中的图片引用

#### 方案 B：使用 SVG 矢量图（最佳）
1. 将 LOGO 转换为 SVG 格式
2. SVG 可以完全控制颜色，且无损缩放
3. 文件更小，加载更快

#### 方案 C：继续优化 CSS（临时）
当前已应用的 CSS：
```css
.logo-img {
  background: white !important;
  filter: saturate(0.7) brightness(1.1) contrast(1.1);
  mix-blend-mode: multiply;
}
```

### 验证方法
1. 用图片查看器打开 LOGO.jpg
2. 检查图片本身是否有淡蓝色背景
3. 如果有，必须替换图片文件

## 问题 2：英文版公司名称未显示为 "Talenturbo"

### 已实施的修复
1. ✅ 移除了 footer-name 的 data-i18n 属性
2. ✅ 添加了 updateCompanyName() 函数
3. ✅ 在页面加载和语言切换时调用该函数
4. ✅ 将函数设为全局可访问：`window.updateCompanyName`

### 验证步骤
1. 打开浏览器开发者工具（F12）
2. 在控制台输入：`window.ExquisysI18n.lang`
3. 应该显示 'zh' 或 'en'
4. 切换语言后，检查 footer-name 的文本内容

### 如果仍未生效
1. 清除浏览器缓存
2. 清除 localStorage：在控制台输入 `localStorage.clear()`
3. 硬刷新页面（Ctrl+F5）
4. 检查控制台是否有 JavaScript 错误

## 问题 3：英文版语言按钮未显示 "CN"

### 已实施的修复
修改了 applyI18n() 函数中的按钮文本逻辑：
```javascript
btn.textContent = window.ExquisysI18n.lang === 'zh' ? 'EN' : 'CN';
```

### 验证步骤
1. 打开页面（默认中文），按钮应显示 "EN"
2. 点击按钮切换到英文
3. 按钮应立即变为 "CN"
4. 再次点击，应切换回中文，按钮显示 "EN"

### 如果仍未生效
1. 检查浏览器控制台是否有错误
2. 确认 js/main.js 和 js/i18n.js 已正确加载
3. 在控制台输入：`window.ExquisysI18n.toggleLang()`
4. 观察按钮文本是否改变

## 通用调试步骤

### 1. 清除缓存
- Chrome/Edge: Ctrl+Shift+Delete
- Firefox: Ctrl+Shift+Delete
- Safari: Cmd+Option+E

### 2. 硬刷新
- Windows: Ctrl+F5 或 Ctrl+Shift+R
- Mac: Cmd+Shift+R

### 3. 检查文件加载
在开发者工具的 Network 标签中确认：
- css/style.css
- js/main.js
- js/i18n.js
都已成功加载（状态码 200）

### 4. 检查 JavaScript 错误
在开发者工具的 Console 标签中查看是否有红色错误信息

### 5. 测试语言切换
在控制台输入以下命令测试：
```javascript
// 查看当前语言
console.log(window.ExquisysI18n.lang);

// 切换语言
window.ExquisysI18n.toggleLang();

// 手动更新公司名称
window.updateCompanyName();
```
