## 调试方法

### 方法1：浏览器开发者工具
1. 在浏览器中打开产品页面
2. 按F12打开开发者工具
3. 在Console中输入以下代码来显示热区边框：

```javascript
document.querySelectorAll('.product-hotspot').forEach(el => {
  el.style.border = '2px solid red';
  el.style.background = 'rgba(255, 0, 0, 0.2)';
});
```

### 方法2：临时CSS调试
在 `css/style.css` 中临时添加（调试完成后删除）：

```css
.product-hotspot {
  border: 2px solid red !important;
  background: rgba(255, 0, 0, 0.2) !important;
}

.product-hotspot::before {
  border: 2px dashed blue !important;
  background: rgba(0, 0, 255, 0.1) !important;
}
```

红色边框 = 热区外边界
蓝色虚线 = 实际交互区域（90%居中）

## 调整建议

如果热区位置不准确，需要根据实际图片调整：

### 水平位置调整
- 向左移动：减小 `left` 值
- 向右移动：增加 `left` 值
- 调整间距：修改相邻热区的 `left` 差值

### 垂直位置调整
- 向上移动：减小 `top` 值
- 向下移动：增加 `top` 值

### 尺寸调整
- 扩大热区：增加 `width` 和 `height`
- 缩小热区：减小 `width` 和 `height`

## 精确定位步骤

1. 在图片编辑软件中打开 `Talenturbo_Product_Overview.png`
2. 测量每个产品框的实际位置（像素坐标）
3. 获取图片总尺寸
4. 计算百分比：
   - `left% = (框左边距 / 图片宽度) × 100`
   - `top% = (框上边距 / 图片高度) × 100`
   - `width% = (框宽度 / 图片宽度) × 100`
   - `height% = (框高度 / 图片高度) × 100`

## 常见问题

### 问题1：热区偏移
- 原因：图片在不同屏幕尺寸下缩放
- 解决：使用百分比定位（已实现）

### 问题2：热区太小/太大
- 原因：产品框在图片中的实际大小与预估不符
- 解决：调整 `width` 和 `height` 值

### 问题3：点击无响应
- 原因：热区被其他元素遮挡或 `z-index` 太低
- 解决：检查 CSS 中的 `z-index` 设置（当前为10）

### 问题4：移动端误触
- 解决：已通过 `pointer-events: none` 禁用移动端热区

## 推荐的调试流程

1. 添加临时CSS显示热区边框
2. 在浏览器中查看热区是否对齐产品框
3. 使用浏览器开发者工具实时调整位置
4. 记录最佳位置参数
5. 更新 `products.html` 中的热区坐标
6. 删除临时CSS
7. 测试所有热区的悬停和点击功能
