# 功能测试清单

## 1. LOGO 背景色测试
- [ ] 打开任意页面，检查左上角 LOGO 是否还有淡蓝色背景
- [ ] 如果仍有背景色，说明是图片本身的问题，需要替换图片文件
- [ ] 当前已应用 CSS 滤镜：`filter: saturate(0.8) brightness(1.05); mix-blend-mode: darken;`

## 2. 语言切换按钮测试
### 中文版（默认）
- [ ] 打开页面，语言按钮应显示 "EN"
- [ ] 点击按钮切换到英文版

### 英文版
- [ ] 语言按钮应显示 "CN"
- [ ] 点击按钮切换回中文版

## 3. 公司名称显示测试
### 中文版
- [ ] 页面底部应显示："北京玄码智创信息技术有限公司 · Talenturbo"

### 英文版
- [ ] 页面底部应显示："Talenturbo"（仅显示英文名）

## 4. 移动端菜单测试
- [ ] 在手机浏览器或开发者工具的移动模式下打开
- [ ] 点击汉堡菜单（三条横线）
- [ ] 菜单应从左侧滑出，显示所有5个选项：
  - 产品
  - 专业服务
  - 成功案例
  - 合作伙伴
  - 关于
- [ ] 点击遮罩层或菜单项，菜单应关闭

## 5. LOGO 尺寸测试
### 桌面端
- [ ] LOGO 高度应为 192px（比原来的 120px 大 60%）

### 平板端（1024px 以下）
- [ ] LOGO 高度应为 144px

### 移动端（768px 以下）
- [ ] LOGO 高度应为 96px

## 调试提示
如果功能未生效，请：
1. 清除浏览器缓存（Ctrl+Shift+Delete 或 Cmd+Shift+Delete）
2. 硬刷新页面（Ctrl+F5 或 Cmd+Shift+R）
3. 打开浏览器开发者工具（F12）查看控制台是否有错误
4. 检查 localStorage 中的 'exquisys-lang' 值
