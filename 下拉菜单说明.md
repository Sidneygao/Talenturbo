## 功能特点

### 桌面端
- **悬停显示**：鼠标悬停在"产品"、"专业服务"、"关于"上时自动显示下拉菜单
- **平滑动画**：下拉菜单带有淡入和向下滑动的动画效果
- **视觉反馈**：
  - 菜单项悬停时显示下划线动画
  - 下拉项悬停时左侧显示蓝色边框
  - 背景色变化提示当前位置

### 移动端
- **点击展开**：点击带下拉菜单的项目展开子菜单
- **手风琴效果**：展开一个菜单时，其他菜单保持状态
- **易于操作**：
  - 子菜单项缩进显示
  - 点击子菜单项后自动关闭侧边栏
  - 点击遮罩层关闭菜单

## 样式设计

### 下拉菜单样式
```css
- 最小宽度：280px
- 背景：白色
- 边框：1px 浅灰色
- 圆角：8px
- 阴影：0 8px 24px rgba(0, 0, 0, 0.12)
```

### 下拉项样式
```css
- 内边距：0.75rem 1.25rem
- 左边框：3px 透明（悬停时变为蓝色）
- 标题：粗体 0.95rem
- 描述：灰色 0.85rem
```

## 双语支持

所有菜单项都支持中英文切换：
- 使用 `data-i18n` 属性标记需要翻译的文本
- 切换语言时自动更新所有菜单文本
- 包括下拉菜单中的标题和描述

## 锚点链接

下拉菜单项使用锚点链接，方便直接跳转到页面特定部分：
- `products.html#hrcore` - 跳转到产品页面的 HR CORE 部分
- `services.html#framework` - 跳转到服务页面的框架设计部分
- `about.html#management` - 跳转到关于页面的管理团队部分

```css
.dropdown-menu {
  /* 调整下拉菜单样式 */
}

.dropdown-item {
  /* 调整下拉项样式 */
}
```
